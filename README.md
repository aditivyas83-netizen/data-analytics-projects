## 1️⃣ Consumer Price Index (CPI) Analysis – Excel Case Study

A detailed Excel-based analysis of the Consumer Price Index (CPI) data to identify inflation trends and insights for economic decision-making.

### Key Features
- Cleaned and organized CPI datasets for accurate analysis.
- Calculated month-on-month and year-on-year CPI changes.
- Used Excel formulas like SUM, AVERAGE, COUNTIF, and VLOOKUP for data analysis.
- Created pivot tables and charts to visualize inflation trends across categories.
- Highlighted key inflation drivers and trends over time.
- Generated actionable insights for economic planning and pricing strategy.

### Tools & Skills
- Microsoft Excel (Formulas, Pivot Tables, Charts)
- Data Cleaning & Analysis
- Trend Analysis & Visualization

## 4️⃣ E-commerce Orders Dashboard – Power BI

Developed an interactive Power BI dashboard to analyze e-commerce orders, track KPIs, and generate actionable business insights.

### Key Features
- Interactive filters for Brand, Platform, Store, Date, Product Category, and Discount.
- Built KPIs for Revenue, Orders, Average Order Value, and Customer Segments (New vs Returning).
- Hierarchical sales breakdown: Brand → Platform → Store.
- Month-on-month trend analysis to guide pricing and promotional strategies.
- Dynamic visualizations for quick decision-making.

### Tools & Skills
- Power BI (DAX, Power Query)
- Data Visualization & Dashboarding
- KPI Analysis & Business Insights
